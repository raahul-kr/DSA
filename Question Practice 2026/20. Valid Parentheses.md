# Valid Anagram

This repository contains a C++ solution for the classic **Valid Anagram** problem (LeetCode 242).

## 📝 Problem Description
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise. 

An **anagram** is a word formed by rearranging the letters of a different word, using all the original letters exactly once. For example, "listen" and "silent" are anagrams.

---

## 💡 Intuition and Approach
The most reliable way to check for an anagram is to count the frequency of each letter in both strings. If both strings have the exact same amount of each letter, they are anagrams. 

We can achieve this using a **Frequency Counter** (Hash Map). 
1. We add to the count for every letter in the first string.
2. We subtract from the count for every letter in the second string.
3. If the strings are anagrams, every letter's count will perfectly balance out to `0`.

---

## 🚀 C++ Source Code

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    bool isAnagram(string s, string t) {
        // Step 1: Fast fail if lengths differ
        if (s.size() != t.size()) return false;

        // Step 2: Create a hash map to store character frequencies
        unordered_map<char, int> count;

        // Step 3: Increment for string 's', decrement for string 't'
        for (char c : s) count[c]++;
        for (char c : t) count[c]--;

        // Step 4: Verify all counts are zero
        for (auto const& [key, val] : count) {
            if (val != 0) return false;
        }

        return true;
    }
};
